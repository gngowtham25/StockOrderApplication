Prerequisites :
--------------

Environment:
Download and install Java Development Kit SE 8 (Java Version : 1.8.0_131)

Application:
Stock Order Details should be present in StockOrderList.csv file.
All characters are case sensitive and fields are separated using a space or tab.

===================================================================================

Steps to run the application


1) Run the following command to start the application from Application root folder.

    sh start.sh prod

====================================================================================



For modifying Java files, open the project using IntelliJ IDEA, make necessary changes
and run StockOrderApplication configuration.
